Dynamical systems


  • A continuous dynamical system is a pair $$(X, \phi)$$ where $\mathcal{M}$ is a topological space and $\phi \colon \mathbb{R} \times \mathcal{X} \to X$ is a continuous map such that $\phi(0, p) = p$ and $\phi(s, \phi(ùë°, ùëù)) = \phi(s + t, p)$ for all $p \in X$ and all $t, s \in \mathbb{R}$.

  • The typical examples arising from differential equations have as state space a smooth manifold $\mathcal{M}$ and the dynamics are given by the integral curves of a smooth vector field on $X$.

  • A set $A \subset X$ is called an attractor if it satisfies the following conditions: it is compact, it is an invariant set (i.e. if $a \in A$ then $\phi(ùë°, ùëé) \in ùê¥$ for all $ùë° \geq 0$) and it has an open basin of attraction. Equivalently, there is an invariant open neighborhood $ùëà \subset \mathcal{M}$ of $ùê¥$ such that $\cap_{t\geq 0} \{\phi(ùë°, ùëù) ‚à∂ ùëù \in ùëà\} = ùê¥$.

Geometry of the attractor

  • Given a topological space $X$, its homology $H_d(X)$ at degree $d$ captures geometric signatures in $X$ of dimension $d$.
  • The rank of the group $H_d(X)$ defines the $d$-th Betti number $b_d$ of $X$ and it quantifies the number of (independent) $k$-dimensional holes in $X$.

    • $b_0$ = number of connected components,
    • $b_1$ = number of 1-dimensional cycles,
    • $b_2$ = number of 2-dimensional voids, $\dots$



      • $b_0$ = 1
      • $b_1$ = 2
      • $b_2$ = 0

In practice, we only have measurements, that are the result of an observation function $F:\mathcal{M}\to \mathbb R$ and an initial state $p\in \mathcal{M}$ that produces a time series \begin{align}\varphi_p:\mathbb{R}&\to \mathbb{R}\\ t&\mapsto F(\phi(t,p)) \end{align}

Theorem (Takens). Let $\mathcal{M}$ be a smooth, compact, Riemannian manifold. Let $\tau > 0$ be a real number and let $d ‚â• 2 \mathrm{dim}(\mathcal{M})$ be an integer. Then, for generic $\phi \in C^2(\mathbb{R} \times \mathcal{M}, \mathcal{M})$ and $F\in C^2(\mathcal{M}, \mathbb{R})$ and for $\varphi_\bullet(ùë°)$ defined as above, the delay map \begin{align} \varphi~ \colon & ~~\mathcal{M} &\rightarrow & ~~\mathbb{R}^{d+1}\\ &~~p &\mapsto & ~~(\varphi_p(0), \varphi_p(ùúè), \varphi_p(2ùúè),\dots, \varphi_p(d\tau)) \end{align} is an embedding (i.e., $\varphi$ is injective and its derivative has full-rank everywhere).

From tume series to topology


  • Let $f\colon\mathbb{R} \rightarrow \mathbb{R}$ be a function, $\tau> 0$ a real number and $d > 0$ an integer. The time delay embedding of $f$ with parameters $d$ and $\tau$ is the vector-valued function \begin{align} X_{d,\tau}f\colon &~~\mathbb{R}\rightarrow ~~\mathbb{R}^{d+1}\\ &~~~t ~\mapsto ~(f(t), ùëì(t + \tau), ùëì(t + 2\tau), \dots, ùëì(t + d\tau))\\ \end{align}
  • Given time series data $f(t) = \varphi_p(t)$ with $t\in T$ observed from a potentially unknown dynamical system $(X, \phi)$, Takens‚Äô theorem implies that (generically) the time delay embedding $X_{d,\tau}f([0,T])$ provides a topological copy of $\{\phi(t, p) ‚à∂ t \in T\} \subseteq X$.
    In particular this method reconstructs attractors.

    Given $X_1, X_2, \dots X_N:[0,T]\to \mathbb{R}$ signals, we construct an embedding in $\mathbb{R}^N$ given by $$\{(X_1(t), X_2(t), \dots, X_N(t)): t\in [0,T]\}$$

    Sliding Window Embedding

    • Let $X_1, X_2, \dots X_N:[0,T]\to \mathbb{R}$ a set of signals.
    • Given a window size $W$, we compute for every $t\in [W,T]$ the embedding of $X_1, X_2, \dots X_N:[t-W, t]\to \mathbb{R}$ in $\mathbb{R}^N$.
